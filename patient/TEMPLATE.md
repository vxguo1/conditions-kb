# Patient Layer — Private Templates

> **🔒 PRIVATE — NEVER COMMIT THIS FOLDER.**
> Everything in `patient/` is git-ignored **except this template file**. This is
> where a real person's health information would live, so it stays on this
> machine only — it is never committed to git and never pushed anywhere.
>
> **How to use:** copy this file to a new, private file (e.g.
> `patient/<condition>-notes.md`, which git ignores) and fill it in there. Do
> **not** put real personal or medical details into `TEMPLATE.md` itself — keep
> this one blank as a reference.
>
> This is an organizing aid, **not medical advice**. Nothing here replaces the
> care team. In an emergency, contact a doctor or emergency services.

This repo is a multi-condition knowledge base, so the same template works for any
condition (diabetes, heart disease, kidney disease, and so on). Copy it once per
condition or person as needed.
